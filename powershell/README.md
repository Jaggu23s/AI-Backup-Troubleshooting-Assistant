# Unified PowerShell Collector

The Unified PowerShell Collector is responsible for gathering diagnostic information required for AI-assisted investigations.

Instead of asking engineers to execute multiple commands manually, the collector automatically gathers relevant evidence based on the identified issue category.

---

## Objectives

- Standardize diagnostic collection
- Reduce manual effort
- Collect consistent evidence
- Support AI-assisted analysis

---

## Collector Modules

- Backup
- Windows
- Network
- Storage
- VMware
- Performance

Each module can be executed independently or as part of a combined investigation.

---

## Workflow

Issue Detected

↓

Determine Required Modules

↓

Execute Unified Collector

↓

Upload Collected Evidence

↓

AI Investigation

↓

Root Cause Analysis
